# 水果销售系统 python876

## 项目概述

水果销售系统是一个基于Python语言的Web应用，融合了Django后端框架与Vue.js前端框架，旨在为不同角色用户提供一个高效、易用的在线水果购买与管理系统。

## 技术栈

- 后端：Python 3.7，Django框架
- 前端：Vue.js
- 数据库：MySQL
- 开发工具：PyCharm

## 功能模块

### 系统角色

#### 管理员

- 首页
- 个人中心
- 门店管理
- 用户管理
- 商品分类管理
- 商品信息管理
- 商品咨询管理
- 系统管理

#### 用户

- 首页
- 个人中心
- 商品咨询管理
- 我的收藏管理

#### 门店

- 首页
- 个人中心
- 商品信息管理
- 商品咨询管理
- 订单管理
